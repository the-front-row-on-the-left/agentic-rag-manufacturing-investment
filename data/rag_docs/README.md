이 디렉터리에 PDF 문서를 넣고 `manifest.json`을 직접 작성하세요.

예시:
- doc1_oecd_ai_manufacturing.pdf
- doc2_nist_mep_ai_implementation.pdf
- doc3_world_manufacturing_report_2024.pdf
- doc4_industrial_ai_market_report.pdf

권장 `manifest.json` 항목:
- `doc_id`
- `filename`
- `title`
- `issuer`
- `year`
- `source_type`
- `tags`
